# ✅ **TABEL FITUR ADMIN (Tanpa Kolom Progres)**

### **FITUR ADMIN**

| Penanggung jawab | Fitur                                                                                  |
| ---------------- | -------------------------------------------------------------------------------------- |
| **SIGIT**        | Manajemen buku (tambah, edit, hapus, kategori, stok, lokasi rak)                       |
|                  | Manajemen peminjaman & pengembalian (validasi peminjaman, pengembalian, update status) |
|                  | Manajemen e-book (unggah file e-book dan hubungkan ke data buku)                       |
| **ROKY**         | Manajemen denda (perhitungan otomatis denda keterlambatan)                             |
|                  | Verifikasi pembayaran (cek bukti pembayaran QRIS dari mahasiswa)                       |
|                  | Laporan sistem (laporan peminjaman, denda, statistik buku populer)                     |
|                  | Manajemen rating (menampilkan rating buku pada halaman admin)                          |

# ✅ **TABEL FITUR MAHASISWA (Tanpa Kolom Progres)**

### **FITUR MAHASISWA**

| Penanggung jawab | Fitur                                                 |
| ---------------- | ----------------------------------------------------- |
| **BANGKIT**      | Registrasi & Login                                    |
|                  | Melihat katalog buku (judul, penulis, kategori, stok) |
|                  | Pencarian buku (berdasarkan judul, penulis, kategori) |
| **ADI**          | Peminjaman buku fisik                                 |
|                  | Peminjaman e-book (unduh e-book dari sistem)          |
|                  | Informasi lokasi rak buku (gedung, lantai, kode rak)  |
| **NUN**          | Riwayat peminjaman (pinjaman aktif & riwayat)         |
|                  | Pembayaran denda via QRIS                             |
|                  | Rating & ulasan buku                                  |
